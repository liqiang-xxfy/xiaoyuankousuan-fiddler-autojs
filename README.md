
##### 小猿口算自动PK刷题，该项目仅供娱乐


## 功能

  -  可以修改PK答题时间为0.001秒
  
  -  取消过渡动画，3秒一题，可以快速刷分
  
  -  自动答题，无需绘制答案

## 工具

 - fiddler-everywhere   ---需要电脑，要求会安装证书
  
 - AutoX.js  ---需要安卓手机或者模拟器

## 思路

1.使用fiddler-everywhere抓包，正则替换修改js，实现取消过渡动画、修改答题时间0.001秒、快速自动答题

2.AutoX.js点击开始按钮重复答题
