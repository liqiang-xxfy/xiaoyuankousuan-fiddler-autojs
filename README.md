## xiaoyuankousuan-fiddler-autojs
#### 小猿口算自动PK刷题，该项目仅供娱乐


## 功能

  ### 1. 可以修改PK答题时间为0.001秒
  
  ### 2. 取消过渡动画，3秒一题，可以快速刷分
  


## 工具

  fiddler-everywhere   //需要电脑，要求会安装证书
  
  AutoX.js  //需要安卓手机或者模拟器

## 思路

1.使用fiddler-everywhere抓包，正则替换修改js，实现取消过渡动画、快速自动答题

2.AutoX.js点击开始按钮重复答题
